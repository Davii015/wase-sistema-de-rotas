# wase-sistema-de-rotas
Wase é um sistema de simulação de rotas urbanas que permite visualizar e calcular caminhos entre diferentes pontos de uma cidade. Utilizando representação em grafos e interface interativa desenvolvida em React e JavaScript, o projeto demonstra conceitos de navegação, conexões entre locais e planejamento de trajetos para fins educacionais.

Como rodar?

Abra o CMD
Abra a pasta "sistema-rotas"

cd "C:\Users\davit\OneDrive\Área de Trabalho\projeto-rotas\projeto-rotas\sistema-rotas"

npm install

npm run dev

(Obs: Tenha o node.js instalado)
